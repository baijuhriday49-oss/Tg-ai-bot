STEP 01

Create your Telegram bot

Open Telegram → search @BotFather → send /newbot → give it a name → copy the token it gives you.

💡 Takes literally 30 seconds. BotFather is the official Telegram bot.
STEP 02

Get your Groq API key

Go to console.groq.com → sign up free → create an API key. Used for both chat replies and voice transcription (Whisper).

STEP 03

Install ffmpeg in Termux

Needed to convert Telegram voice messages (ogg) before transcribing.

pkg install ffmpeg
STEP 04

Set up the project

mkdir tg-ai-bot && cd tg-ai-bot # drop bot.js, package.json, .env.example here npm install cp .env.example .env nano .env # paste your TELEGRAM_BOT_TOKEN and GROQ_API_KEY
STEP 05

Run it & chat! 🚀

node bot.js

Open your bot on Telegram → send /start → start chatting or send a voice note!

✅ No ngrok needed. No webhook. Just runs and listens via polling. Simple.
